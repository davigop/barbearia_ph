Projeto Front-End – Barbearia PH

Este projeto foi desenvolvido como atividade de front-end com o objetivo de criar um site responsivo para uma barbearia fictícia/real, aplicando conceitos de HTML, CSS e responsividade (mobile-first).

Tecnologias utilizadas
HTML5
CSS3
Bootstrap
JavaScript 

Objetivo do projeto:
Criar um site moderno e responsivo para a Barbearia PH, com foco em apresentação dos serviços, identidade visual e facilidade de contato através do WhatsApp.

Estrutura do site:
O site é composto pelas seguintes páginas:
Página inicial (Home)
Página de serviços

Equipe
Davi: Desenvolvimento principal do projeto, incluindo HTML, CSS, JavaScript, organização do layout, responsividade e planejamento geral.
Gabriel: Contribuiu com a construção do HTML, revisão do código, testes de responsividade e sugestões de melhorias na interface.

Funcionalidades:
Layout responsivo para dispositivos móveis e desktop
Seção de serviços
Botão de contato via WhatsApp
Interface simples e intuitiva

Observação:
Este projeto foi desenvolvido com fins acadêmicos.
